# AstroAI - AI-Powered Vedic Astrology Platform

## Project Structure
- `/frontend` - React Native (Expo) app for Web & Android
- `/backend` - Firebase Cloud Functions
- `/astrology-engine` - Python calculation engine
- `/database` - Firestore schema & rules
- `/docs` - Documentation

## Setup
See SETUP_GUIDE.md in /docs folder

## Development
- Frontend: `cd frontend && npm start`
- Backend: `cd backend/functions && npm run serve`
- Python: `cd astrology-engine && python main.py`
